# DirScanner

**DirScanner** - це легка та ефективна кросплатформна консольна утиліта на C#, яка допомагає швидко зібрати назви всіх файлів із певним розширенням у текстовий звіт. Більше не потрібно копіювати назви вручну!

## Особливості

- **Кросплатформність**: Працює на Windows, macOS та Linux.
- **Нативні діалоги**: Використовує системні вікна для вибору папок та збереження файлів (забудь про "ромбики" та проблеми з кодуванням).
- **Розумний сканер**: Автоматично знаходить усі унікальні розширення файлів у обраній директорії.
- **Чистий звіт**: Генерує текстовий файл із "шапкою", що містить дату сканування та шлях до джерела.
- **Швидкість**: Сканує тільки поточну директорію, не перевантажуючи систему підпапками.

## Як користуватися

1. **Запуск**: Запустіть програму через вашу IDE або термінал (`dotnet run`).
2. **Вибір папки**: У вікні, що з'явиться, оберіть директорію для аналізу.
3. **Фільтрація**: У консолі відобразиться список усіх знайдених розширень. Введіть номер потрібного.
4. **Збереження**: Оберіть місце та назву для вашого `.txt` звіту.

## Технологічний стек

- **Мова**: C#
- **Платформа**: .NET 10.0
- **Бібліотеки**: Windows Forms (для Windows), AppleScript (для macOS), Zenity (для Linux).
